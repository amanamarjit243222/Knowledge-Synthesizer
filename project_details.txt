===========================================================
    🧠 KNOWLEDGE SYNTHESIZER: AI-POWERED LEARNING PIPELINE
===========================================================
Team: Team RTA
Mission: Transforming passive listening into active, retained expertise.

-----------------------------------------------------------
1. THE PROBLEM: LOST INSIGHTS
-----------------------------------------------------------
*   Classroom discussions and meetings are goldmines of insight, but they often evaporate once the session ends.
*   Valuable arguments get lost in the noise.
*   Impossible to track how ideas evolved or who said what.
*   Lack of retention: Dynamic debates become fleeting memories rather than lasting learning resources.

-----------------------------------------------------------
2. THE SOLUTION: AI SYNTHESIS
-----------------------------------------------------------
*   An intelligent audio-to-knowledge system.
*   Captures unstructured discussions and converts them into clear, interactive learning maps.
*   Filters out noise and identifies core logic.
*   Maps arguments to counterarguments and highlights open questions.

-----------------------------------------------------------
3. CORE TECHNICAL FEATURES
-----------------------------------------------------------
*   🚀 SMART TRANSCRIPTION: Seamlessly converts speech to text using SpeechRecognition and pydub.
*   🧠 NLP INSIGHTS: Automatically extracts key arguments and builds Concept Maps via TextBlob.
*   📊 INTERACTIVE KNOWLEDGE GRAPH: Visualizes the flow of discussion with draggable nodes for reorganization.
*   📱 DUAL ACCESS MODES:
    - Web Server Mode: Scalable FastAPI-driven experience.
    - Local File Mode: Runs directly as a local index.html (no server required).
*   🎭 PREMIUM UI: Modern "Deeptech" aesthetics with glassmorphism, dynamic animations, and dual theme support.

-----------------------------------------------------------
4. TECHNOLOGY STACK
-----------------------------------------------------------
*   BACKEND: Python 3.10+, FastAPI (Asynchronous Processing)
*   NLP: TextBlob (Sentiment & Polarity Analysis)
*   AUDIO: speech_recognition, pydub (FFmpeg backend)
*   FRONTEND: Vanilla HTML5, CSS3 (Custom Design System), JavaScript (ES6)
*   VISUALIZATION: HTML5 Canvas API (Custom Graph Rendering Engine)

-----------------------------------------------------------
5. HOW IT WORKS (THE WORKFLOW)
-----------------------------------------------------------
Step 1: CAPTURE INPUT (Live Audio, Transcript, or Voice File).
Step 2: SET CONTEXT (Roundtable, Survey, or Debate mode).
Step 3: AI SYNTHESIS (Engine extracts keynotes, generates summaries, and constructs maps).
Step 4: VISUAL INSIGHT (User receives a structured, interactive Knowledge Graph).

-----------------------------------------------------------
6. KEY BENEFITS
-----------------------------------------------------------
*   Deepens comprehension through visual logic mapping.
*   Allows rapid review of hour-long sessions in minutes.
*   Identifies consensus and pinpoint unresolved friction.
*   Bridges the gap between "Information" and "Retention".
